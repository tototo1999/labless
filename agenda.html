共识实验室 · 研究纲领

我们在测什么 · 九大科室

AI 圈的流言不止一类。我们把它们归成九个科室,每科一类追问,每条都奔着可验证去——无实验不盖章。这页是地图,不是结论;开庭的去悬案馆看卷宗,在池的等点火。

共识是跑出来的,不是聊出来的
①已开庭

能力真伪

Capability
  • 裸模型连多位数四则都算不对?
  • 给它一个计算器,就天下无敌了吗?
  • 同一副工具,何时是解药、何时是税?
②活案·每日

降质稳定

Degradation
  • "又降智了"——真掉智商,还是你客户端/中转的锅?
  • 厂商偷换量化版、换便宜 provider,抓得到吗?
  • 同一个模型,今天比上周差多少?
③已开庭

多 Agent 社会

Multi-agent Society
  • 一群 AI 共享一个资源,守得住不作死吗?
  • 放进自治小镇,会相爱还是会纵火?
  • 有内鬼带头作弊,其他 AI 跟风还是制裁?
④排队中

幻觉与诚实

Hallucination & Honesty
  • 会不会一本正经地编出处、编数据?
  • 没人看着的时候,它会作弊吗?
  • 嘴上说得最满的,手上做得最差吗?
⑤在池

安全与供应链

Supply-chain Security
  • 装个第三方 skill,会把你的 API key 偷走吗?
  • 接的工具/MCP,会不会被投毒、被注入?
  • 藏在网页里的一句话,能劫持你的 agent 吗?
⑥在测

成本与玄学

Cost & Folklore
  • 贵的模型,真的更稳定吗?
  • "深呼吸""一步步想"这类玄学 prompt 真有用?
  • 技能包动辄几十万字,这上下文税值不值?
⑦在池

对齐与人格

Alignment & Persona
  • AI 老道歉,是真知错还是嘴上认错?
  • 被一吓就改答案——有主见还是只会谄媚?
  • 不同模型有没有固定"性格"(固执/冒进/老好人)?
⑧在池

记忆

Memory
  • 聊着聊着就忘了?埋个事实,N 轮后还记得吗?
  • memory 工具是真记住,还是装样子记住?
  • 它"记得"的,是真事还是它编的(记忆幻觉)?
⑨在池

破坏力与安全边界

Destructiveness & Safety Boundary
  • "为了安全我不能"——真做不到,还是话术?
  • 声称禁用的能力,换个说法会漏出来吗?
  • "不会"和"不肯"的真实分界线在哪?
